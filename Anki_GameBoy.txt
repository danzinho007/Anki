Em que ano foi lançado o primeiro Game Boy?	1989      
Quem criou o Game Boy?	Gunpei Yokoi      
Qual era o tipo de tela usado no Game Boy original?	Tela LCD em preto e branco      
Quantas pilhas AA eram necessárias para alimentar o Game Boy original?	Quatro pilhas AA      
Qual é o nome do jogo de quebra-cabeça mais famoso lançado junto com o Game Boy?	Tetris      
Qual foi o sucessor direto do Game Boy original?	Game Boy Color      
Qual acessório permitia conectar o Game Boy ao Super Nintendo?	Super Game Boy      
Quantas cores a tela do Game Boy Color podia exibir simultaneamente?	Até 56 cores na tela (de uma paleta de 32.768)      
Qual foi o último modelo da linha Game Boy lançado pela Nintendo?	Game Boy Micro      
Qual jogo de Pokémon foi um dos mais populares no Game Boy?	Pokémon Red e Blue
